# Fuentes para PDF (Consultoría Premium)

Copiá estos **4 archivos** desde la carpeta descargada `static/` de Noto Sans (NO variable fonts) a `public/fonts/`:

- **NotoSans-Regular.ttf**
- **NotoSans-Bold.ttf**
- **NotoSans-Italic.ttf**
- **NotoSans-BoldItalic.ttf**

Descarga: [Google Fonts – Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans) → Download family → descomprimí el zip → en la carpeta `static/` copiá los 4 archivos anteriores (no uses las variable fonts de la carpeta `variable/`).

Resultado esperado en el proyecto:

```
public/fonts/
  NotoSans-Regular.ttf
  NotoSans-Bold.ttf
  NotoSans-Italic.ttf
  NotoSans-BoldItalic.ttf
```
